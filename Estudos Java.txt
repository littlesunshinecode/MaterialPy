PROCESSAMENTO E CASTING
1. Exemplo de atribuição:
int x, y;
                x = 5;
                y = 2 * x;
                
                System.out.println(x);
                System.out.println(y);
CONSOLE: 5 / 10


 
2. Exemplo de atrib. com ponto flutuante (double/.0):


int x;
                double y;
                x = 5;
                
                y = 2 * x;
                
                System.out.println(x);
                System.out.println(y);
CONSOLE: 5 / 10.0








3. Exemplo de outra atribuição.


int a, b;
double resultado;


a = 5;
b = 2;


resultado = a / b;


System.out.println(resultado);


CONSOLE: 2.0 


motivo: a e b são números inteiros e o java entende, sem o código necessário, como uma divisão inteira.


como resolver: adicionar (double) antes da expressão que resulta na função CASTING (conversão explícita dos valores).


ou seja, se:
int a, b;
double resultado;
a = 5;
b = 2;


resultado = (double) a / b;


System.out.println(resultado);


no CONSOLE vai aparecer: 2.5

1. Soma
Faça um programa para ler dois valores inteiros, e depois mostrar na tela a soma desses números com uma mensagem explicativa, conforme exemplos. 


import java.util.Scanner;
public class MainSoma {
        public static void main(String[] args) {        
                Scanner sc = new Scanner(System.in);


                int A, B, soma;


                A = sc.nextInt();
                B = sc.nextInt();


                soma = A + B;


                System.out.println("SOMA = " + soma);


                sc.close();
        }
}
